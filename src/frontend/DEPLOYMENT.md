# Deployment Guide

This document describes the release and rollback procedures for the learning& Earning application.

## Version Management

The application uses a version identifier stored in `frontend/public/app-version.txt` to track which build is currently deployed to production.

### Current Version: Version 2

**Version 2 Features:**
- Core learning platform with lesson categories
- Guided form filling wizard
- Contact/support page
- Scam awareness education
- WhatsApp integration for support
